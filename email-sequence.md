## How to Set This Up

### Option 1: Mailchimp (Free up to 500 contacts)
1. Create account at mailchimp.com
2. Create "Automation" → "Customer Journeys"
3. Set up trigger: "When someone subscribes"
4. Add emails with delays as shown above

### Option 2: ConvertKit (Free up to 1,000 subscribers)
1. Create account at convertkit.com
2. Create "Visual Automations"
3. Set up sequence with the emails above
4. Connect to your landing page form

### Option 3: Brevo (Free up to 300 emails/day)
1. Create account at brevo.com
2. Create "Automation" workflows
3. Set up the sequence

### Option 4: Use FormSubmit + Manual Follow-up (Free, no automation)
1. Form submissions go to your email
2. Copy/paste these templates manually
3. Use Gmail labels to track where people are in sequence

**Recommended: Start with Option 4 (manual), then upgrade to Option 1 once you have 10+ leads.**
