## Automation #1: Email Auto-Responses

**Time Saved:** 3-4 hours/week
**Cost:** Free (Gmail) to $20/month (Gorgias)
**Setup Time:** 30 minutes

**What it does:**
Automatically responds to common questions with pre-written answers.

**Best for:**
- Customer support
- Sales inquiries
- FAQ responses
- Appointment confirmations

**How to set up:**
1. List your 10 most common questions
2. Write responses for each
3. Set up filters in Gmail or use a tool like Gorgias
4. Review weekly and refine

**Example:**
```
Subject: Order Status Inquiry

Hi {{name}},

Thanks for reaching out! Your order #{{order_number}} is currently {{status}}.

Tracking: {{tracking_link}}

Expected delivery: {{delivery_date}}

Questions? Just reply to this email.

Thanks!
```

## Automation #2: Calendar Scheduling

**Time Saved:** 2-3 hours/week
**Cost:** Free (Calendly basic) to $10/month (Calendly pro)
**Setup Time:** 15 minutes

**What it does:**
Eliminates the back-and-forth of scheduling meetings.

**Best for:**
- Sales calls
- Client meetings
- Consultations
- Discovery calls

**How to set up:**
1. Connect Calendly to your calendar
2. Set your availability
3. Add buffer time between meetings
4. Add intake questions
5. Send link instead of proposing times

**Pro tip:** Add screening questions to qualify leads before they book.

## Automation #3: Form to CRM/Email

**Time Saved:** 1-2 hours/week
**Cost:** Free (Google Forms + Zapier free tier)
**Setup Time:** 20 minutes

**What it does:**
Automatically routes form submissions to the right person or system.

**Best for:**
- Contact forms
- Lead capture
- Support tickets
- Applications

**How to set up:**
1. Create form (Typeform, Google Forms, etc.)
2. Connect to Zapier
3. Set up rules: "If X, then send to Y"
4. Add auto-response email

**Example workflow:**
```
Form submitted → Zapier checks fields → 
If "budget > $1000" → Send to sales team + Slack notification
If "budget < $1000" → Send to nurture sequence
```

## Common Mistakes to Avoid

1. **Automating bad processes** - Fix the process first, then automate
2. **Over-automating** - Some things need human touch
3. **Set and forget** - Review automations monthly
4. **Not testing** - Always test before going live
5. **No backup plan** - Have manual process as fallback
